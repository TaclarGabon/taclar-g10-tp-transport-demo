Objectif de la démo

Cette version sert à montrer la logique : un client réserve une place, puis le chauffeur voit la réservation sur son interface si les pages sont ouvertes sur le même navigateur / même domaine GitHub Pages.

Important

Cette version utilise localStorage, donc elle est parfaite pour une démo GitHub Pages. Pour une vraie exploitation avec plusieurs téléphones, il faudra remplacer cette mémoire locale par Firebase.
