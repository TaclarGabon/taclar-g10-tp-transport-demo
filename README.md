TACLAR G10 — Version finale de démonstration

Fonctions principales :
- Réservation client avec nom, téléphone, date, bus, point de montée et nombre de places.
- Tarifs conservés depuis le fichier initial selon le bus, la distance et le point de montée.
- Fermeture automatique des réservations en ligne 30 minutes avant le départ (heure du Gabon).
- Présence client demandée 15 minutes avant l’heure de montée.
- Simulation Airtel Money et ouverture d’un message WhatsApp détaillé vers +1 778 808 2048.
- Interface Chauffeur, Chef d’exploitation et Direction avec rapport quotidien Excel.

Pour tester : extraire complètement le ZIP, puis ouvrir login.html ou index.html.
